# 📝 Full Stack Todo Application

A full-stack Todo application built with **Go (Gin)**, **PostgreSQL**, **Vue 3**, and **Docker**.

---

## 🚀 Tech Stack

### Backend
- Go
- Gin Framework
- PostgreSQL
- pgx driver

### Frontend
- Vue 3
- Vite
- Pinia
- Element Plus

### DevOps
- Docker
- Docker Compose
